## About the project 

Fund me is a smart contract that you can use to raise funds for someone who wants support and demand at least the equivalent of $50 in ETH and you can only withdraw it from the owner. You can see the backend repository [here](https://github.com/pinalikefruit/FundMe)

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/pinalikefruit/FundMe-Frontend.git
   cd FundMe-Frontend
   ```
2. Install Live Server: ExtensionID: ritwickdey.LiveServer in you VisualCode

3. Active button Go Live of this extension

4. Connect you wallet and changes de address of smart contract in constants.js

_Note: You can run your own node with hardhat in the previos fund me project and copy and paste the address, or deploy in testnet._
